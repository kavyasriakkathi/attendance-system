Bulk Upload Students (CSV)

Upload a CSV file to add multiple students and create their logins automatically.

CSV Format Requirements:

The CSV must include these headers (column order doesn't matter). Headers are case-insensitive and can include spaces (e.g. Branch ID).

  • Name: Full name of the student
  • Enrollment: Unique ID (e.g., 2021CSE001)
  • Email: Valid email address
  • Branch_ID: Numeric ID of the branch (e.g., 1 for CSE)

Example header row: name,enrollment,email,branch_id

Note: The password will be set to the last 4 digits of the Enrollment number.
